                                                 HEALTHAXIS
Project Overview
The HealthAxis is a web-based application designed to streamline and automate hospital operations such as patient management, doctor management, and appointment scheduling. The system is developed using Spring Boot and follows a modular and layered architecture to ensure scalability, maintainability, and performance.
Project Objectives
1. To automate hospital management processes.
2. To reduce manual paperwork and human errors.
3. To provide a centralized database for hospital records.
4. To ensure secure access to users.
5. To improve efficiency and data management.
Key Features
- User Authentication and Authorization
- Patient Management Module
- Doctor Management Module
- Appointment Scheduling System
- Search and Filter Records
- Role-based Dashboards
- Database Integration
Tech Stack
Backend: Java
Framework: Spring Boot
Web Framework: Spring MVC
ORM: Hibernate / Spring Data JPA
Database: MySQL
Build Tool: Maven
Frontend: HTML, CSS, Thymeleaf
Version Control: Git & GitHub
